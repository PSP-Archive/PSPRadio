PSP Port Tips:
UP = Previpus Link Down = Next Link
LEFT = Back RIGHT = Forward
O+UP = Page up O+Down = Page Down
L = Prev Window R = Next Window
CROSS = Yes / OK / Enter
SQUARE = No / Cancel
Input mode: L+START = Enter R+START = Exit
START = Context Menu. SELECT = Enter URL.

Hint: Replace APP_Retawq/tips.html with your own html to
override these tips!.
